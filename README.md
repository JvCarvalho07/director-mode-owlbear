# Director Mode v0.6 — Command Center

Extensão para Owlbear Rodeo focada em direção de cena para mestres.

## Novidades da v0.6

- Nova estrutura de **Command Center** com rail lateral e cenas.
- **Comandos de Diretor**: macros que combinam reveal, spotlight, zonas e relógio de ameaça.
- Preview visual mais próximo do objetivo final.
- Botões para objetivo, pista, zonas, spotlight, ping, ocultar/revelar e snapshots.
- Design mais premium, com dark fantasy, roxos, painéis em vidro e mapa demonstrativo.

## Deploy

```bash
npm install
npm run build
git add .
git commit -m "Director Mode v0.6 command center"
git push
```

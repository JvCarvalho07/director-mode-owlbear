# Director Mode v0.5 — Design System

## Objetivo visual
Interface premium dark-fantasy para parecer uma ferramenta de direção de cena, não um painel genérico.

## Fontes
- Títulos: Cinzel, Georgia, serif
- Corpo/UI: Inter, Segoe UI, system-ui

## Paleta
- Fundo principal: #050712
- Superfície: #101426 / #17112a
- Linha roxa: #3b2a62
- Roxo principal: #8b5cf6
- Violeta brilho: #c084fc
- Azul reveal: #38bdf8
- Vermelho perigo: #ef4444
- Dourado boss/spotlight: #fbbf24
- Verde seguro: #22c55e

## Componentes implementados
- Topbar com marca, status de conexão e ações rápidas.
- Cards glass/dark fantasy com borda violeta.
- Botões de estado com cor por modo.
- Preview visual de battlemap, reveal ring, tokens, zonas, objetivo e relógio de ameaça.
- Ícone SVG próprio em `public/icon.svg`.
- Manifest atualizado para v0.5.0.
